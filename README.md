# 🛒 Sistema de Gerenciamento de Produtos (CRUD em Java)

## 📌 Sobre o projeto
Este projeto consiste em um sistema de CRUD (Create, Read, Update, Delete) desenvolvido em Java, com o objetivo de gerenciar produtos de uma loja por meio de um menu interativo no terminal.

A aplicação permite ao usuário cadastrar, listar, atualizar e remover produtos de forma simples e organizada.

---

## 🎯 Objetivo
Aplicar na prática conceitos fundamentais da programação, desenvolvendo lógica, organização de código e pensamento estruturado.

---

## ⚙️ Funcionalidades

O sistema apresenta o seguinte menu:

===== MENU DE OPÇÕES =====  
1 - Cadastrar produto  
2 - Listar produtos  
3 - Atualizar produto  
4 - Remover produto  
5 - Sair  

- **Cadastrar:** adiciona um novo produto  
- **Listar:** exibe todos os produtos cadastrados  
- **Atualizar:** altera os dados de um produto existente  
- **Remover:** exclui um produto do sistema  
- **Sair:** encerra o programa  

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento (uso de atributos privados com getters e setters)
- Manipulação de listas com ArrayList
- Estruturas de controle
- Lógica de programação aplicada na construção do sistema
